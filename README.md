# bark-java-sdk

## Using

```xml

<dependency>
    <groupId>top.misec</groupId>
    <artifactId>bark-java-sdk</artifactId>
    <version>0.0.1</version>
</dependency>
```

## 快速开始

```shell

BarkPush pusher=new BarkPush("https://xxx.xxx.xx","xxxxx");
assertNotNull(pusher);
pusher.simplePush("hello word");

```
